## Pasos para ejecutar
Clonar el proyecto
```
git clone https://github.com/Oscar-CR/RickAndMorty_FlutterWeb.git
```

Instalar dependencias
```
flutter pub get
```

Ejecutar proyecto
```
//Elejir la opcion web (normalmente opcion 2)

flutter run
```
